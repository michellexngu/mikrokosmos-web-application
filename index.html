mikromosmos-blog

A personal web application, that provides a journal entry function and commenting section. Additionally there is a page where I display my projects.

Features

  • Creating journal entries and enter comments
  • Input validation
  • Show a list added journal entries
  • Display of projects

Used technologies

  • Grails Framework

Approach

I started by making a sketch of my application. Then with the use of the course material and the web I started creating a base mvc structure for the project. It was quite difficult at the beginning to generate everything the right way, in order for the IDE to recognize it as the right file. I had quite some difficulties with displaying the journal entries on a seperate page. Fortunately it did work in the end. I had a lot of fun exploring different approaches to present my projects. I am quite satisfied with how it turned out. I did the css styling as I went on. After getting the big chunk done, I got to start with the comment section for the user to interact with.
